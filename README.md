# GazeSelectorForHoloLens
UI for Selection by Gaze made with HoloToolKit and VR-Samples


## Tested Environment
- Unity 2017.4.2f2
- Visual Studio 2017
- HoloLens RS5

## Third party assets
You need to import the following assets.
